# Code to scrape Events CSV 

```
var jlimit = 5, jcounter = 0 

var jInterval = setInterval(() => {
    const target = document.querySelector('[elref="loadMoreButton"]')
    if(target){
        target.click();
        jcounter++;
        if (jcounter === jlimit){
            clearInterval(jInterval)
            alert('All done, you can export CSV now!')
        }
    }
}, 100);
```
